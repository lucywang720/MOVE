# MOVE

It includes the modified Meta-World environment for generating expert demonstration data and the script for training the diffusion model as described in the paper "MOVE: A Simple Motion-Based Data Collection Paradigm for Spatial Generalization in Robotic Manipulation".

## data generation

```
bash generate/scripts/gen_demonstration_metaworld.sh 
```

## diffusion training

```
bash MOVE/diffusion_policy/train.sh
```
